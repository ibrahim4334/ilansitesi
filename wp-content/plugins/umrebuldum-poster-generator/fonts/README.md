# Fonts Dizini

Bu klasöre aşağıdaki font dosyalarını yerleştirin:

## Gerekli Font
- **Inter-Bold.ttf** (Önerilen)

## İndirme Linkleri
- Google Fonts: https://fonts.google.com/specimen/Inter
- Direct: https://github.com/rsms/inter/releases

## Kurulum
1. Inter font ailesini indirin
2. `Inter-Bold.ttf` dosyasını bu klasöre kopyalayın
3. Plugin otomatik olarak bu fontu kullanacaktır

## Fallback
Font dosyası yoksa, plugin sistem fontlarını kullanmaya çalışır:
- Linux: DejaVuSans-Bold.ttf
- Windows: arial.ttf
