# Fonts Dizini

Bu klasörde Inter font ailesinin Bold varyantları bulunmaktadır.

## ✅ Mevcut Fontlar

| Dosya | Boyut | Kullanım |
|-------|-------|----------|
| `Inter_28pt-Bold.ttf` | ~344KB | Büyük posterler (öncelikli) |
| `Inter_24pt-Bold.ttf` | ~344KB | Orta boy posterler |
| `Inter_18pt-Bold.ttf` | ~344KB | Küçük posterler |

## Font Seçim Önceliği

Plugin şu sırayla font arar:
1. `Inter_28pt-Bold.ttf` (En büyük - posterler için ideal)
2. `Inter_24pt-Bold.ttf`
3. `Inter_18pt-Bold.ttf`
4. `Inter-Bold.ttf` (Eski format, backward compat)
5. Sistem fontları (fallback)

## İndirme Linkleri (İhtiyaç halinde)
- Google Fonts: https://fonts.google.com/specimen/Inter
- GitHub: https://github.com/rsms/inter/releases

## Fallback Fontlar
Font dosyası yoksa, plugin sistem fontlarını kullanır:
- Linux: DejaVuSans-Bold.ttf, FreeSansBold.ttf
- Windows: arialbd.ttf, arial.ttf

## Notlar
- Tüm fontlar OFL (Open Font License) altında ücretsizdir
- 28pt versiyon büyük başlıklar için en iyi sonucu verir
- Italic versiyonlar şu an kullanılmamaktadır
